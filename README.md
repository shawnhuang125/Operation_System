# Operation_System
## 介紹
> 駭客必備技能,因為大多數伺服器的操作系統都是基於linux。幾乎等於是駭客領域的半片天。

> 可以用來編寫Windows系統的.bat檔腳本及Linux系統的.sh檔腳本。

> 腳本基本上是將操作指令打在類似記事本上然後儲存成某種副檔名,運行腳本可以自動化的做很多動作。

> 駭客通常會编寫作業系統的腳本來製作成某種病毒然後惡意散播。
