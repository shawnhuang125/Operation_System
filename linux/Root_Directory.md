# Linux根目錄說明
## 總攬

|目錄名稱|主要用途|
|-----|-----|
|`/bin (Binaries)`||
|`/boot`|根開機有關的檔案|
|`/dev (Device Files)`||
|`/etc (Configuration Files)`|系統設定檔都讚這裡,應用程式設定檔也在這裡|
|`/home`|管理員的起始點|
|`/lib (Libraries)`||
|`/media`|插入的媒體裝置,例如USB,光碟....|
|`/mnt (Mount)`||
|`/opt (Optional)`||
|`/proc (Process Information)`||
|`/root`||
|`/run`||
|`/sbin (System Binaries)`||
|`/srv (Service Data)`||
|`/sys (System)`||
|`/tmp (Temporary Files)`|暫存檔放置處,每次開關機後會自動刪除|
|`/usr (User Binaries)`||
|`/var (Variable Files)`|常常會更動的檔案,例如日誌檔|
|`/lost+found`||
